### 5.1：基于多帧序列对齐的多曝光图像融合
> 基于多帧序列对齐的多曝光图像融合研究旨在融合多帧图像，其中多帧图像前景目标相对背景静止，输入序列是具有不同曝光值的图像，通过深度学习技术或者传统算法融合序列图像，最终获取一张视觉效果良好，动态范围高的图像。现针对该任务整理了相关会议期刊论文，减少后续研究者们的检索时间，以供大家开展更深层次的研究。
